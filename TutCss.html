    HOJAS DE ESTILO EN CASCADA

    CSS CSS es un lenguaje de hojas de estilo que permite modificar la apariencia, de manera selectiva, a los elementos de un documento HTML. Por ejemplo, para seleccionar todos los elementos de párrafo en una página HTML y volver el texto dentro de ellos de color rojo, has de escribir el siguiente estilo:
    ANATOMIA DE UNA HOJA DE ESTILO
    Hoja de estilo Conjunto de reglas o estilos
    Regla o estilo Determina la apariencia de uno o mas elementos
    Selector Especifica los elementos que serán afectados por las declaraciones, es decir permite seleccionar elementos a los que se aplicara estilos
    Declaraciones Establece parte del efecto de la regla. Consta de dos partes:

    Propiedad: Cualidad o característica del elemento

    Valor: Especificación precisa de la propiedad.

    		 CSS Color Module
    Usando el modelo RGB

    Podemos establecer el valor de un color mezclando diferentes intensidades de los tres colores primarios.


    		 Codigo de colores
    Usando la función rgb(r,g,b) = rgb(red, green, blue)

    El parámetro red = Especifica la intensidad del rojo y los valores son: [0-255] o [0%-100%]

    El parámetro green = Especifica la intensidad del verde y los valores son: [0-255] o [0%-100%]

    El parámetro blue = Especifica la intensidad del azul y los valores posibles son: [0-255] o [0%-100%]

    		Colores
