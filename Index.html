    ESTRUCTURA BASICA DE UN DOCUMENTO HTML

    Elemento Descripcion Enlace
    DOCTYPE Es una declaracion que le informa al navegador que version de HTML se uso para describir el documento DOCTYPE
    HTML Es un elemento raiz del documento. Todos los demas elementos son descendientes de este elemento HTML
    META El elemento vacio meta proporciona informacion del documento por medio de atributos META
    TITLE Es un elemento que especifica el titulo del documento TITLE
    BODY El elemento body representa el contenido del documento Body

    ELEMENTOS SEMANTICOS

    Elemento Descripcion Enlace
    HEADER El elemnto describe contenido destinado a la parte superior o encabezado de la pagina HEADER
    ARTICLE Este elemento encierra contenido independiente del resto de la pagina web ARTICLE
    SECCION El elemento se utiliza para indicar contenido que tipicamente cubre un tema especifico SECTION
    NAV Es un elemento que representa una seccion de una pagina cuyo proposito es proporcionar enlaces de navegacion. NAV
    ASIDE El elemento representa una parte de un documento cuyo contenido solo esta directamente relacionado con el contenido que le rodea ASIDE
    FOOTER El elemento representa un pie de pagina para su ancestro de la categoria SECTIONING FOOTER
    miManual El elemento representa los contenidos dominantes del documento main
