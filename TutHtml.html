    ESTRUCTURA BASICA DE UN DOCUMENTO HTML

    Elemento Descripcion Enlace
    DOCTYPE Es una declaracion que le informa al navegador que version de HTML se uso para describir el documento DOCTYPE
    HTML Es un elemento raiz del documento. Todos los demas elementos son descendientes de este elemento HTML
    META El elemento vacio meta proporciona informacion del documento por medio de atributos META
    TITLE Es un elemento que especifica el titulo del documento TITLE
    BODY El elemento body representa el contenido del documento Body

    ELEMENTOS SEMANTICOS

    Elemento Descripcion Enlace
    HEADER El elemnto describe contenido destinado a la parte superior o encabezado de la pagina HEADER
    ARTICLE Este elemento encierra contenido independiente del resto de la pagina web ARTICLE
    SECCION El elemento se utiliza para indicar contenido que tipicamente cubre un tema especifico SECTION
    NAV Es un elemento que representa una seccion de una pagina cuyo proposito es proporcionar enlaces de navegacion. NAV
    ASIDE El elemento representa una parte de un documento cuyo contenido solo esta directamente relacionado con el contenido que le rodea ASIDE
    FOOTER El elemento representa un pie de pagina para su ancestro de la categoria SECTIONING FOOTER
    miManual El elemento representa los contenidos dominantes del documento main

    ELEMENTOS HTML

    Elemento Descripcion Enlace
    HTML Representa la raiz del documento html
    h1-h6 proporciona seis niveles de encabezado, el contenido de los elementos se muestran en diferentes tamanios h1-h6
    p Los parrafos se utilizan para dividir un texto largo en bloques de texto p
    br Salto de linea, hace que el navegador avance a la siguiente linea br
    hr Representa una transicion entre parrafos a un tema separado, pero relacionado hr
    wbr Indica al navegador donde podria insertar un salto de linea wbr
    sup Permite definir superindices, muestra exponentes, abreviar numeros ordinales sup
    strong Indica que su contenido tiene una gran importancia, seriedad o urgencia strong
    em Cuando queremos dar enfasis al lenguaje escrito em
    mark Permite resaltar texto de interes MARK mark
    q Indica que el texto encerrado es una cita corta en linea q
    blockquote Indica que el texto contenido es una cita extensa blockquote
    cite Permite incluir el nombre de un trabajo, puede citar una pagina cite
    dfn se utiliza para identificar el termino a definir, se representa en cursiva por parte de los navegadores dfn
    abbr Se utiliza para identificar una abreviatura abbr
    small Permite definir la letra chica y los comentarios secundarios small
    ins Marca partes de un texto que han sido aniadidos al documento ins
    del Marca las partes de un texto que han sido eliminadas del documento del
    s Muestra el texto tachado con una línea que loatraviesa muestra contenido relevante s

    MAS ELEMENTOS

    Elemento Descripcion Atributos Enlace
    table Permite definir una tabla ejemplos
    th Permite definir una celda que será el encabezado de un conjunto de celdas.

    scope: Define el conjunto de celdas que se relacionan con el encabezado.

    row: Establecemos el encabezado de una fila.

    col: Establecemos el encabezado de una columna.

    rowgroup: Establecemos el encabezado de un grupo de filas.

    colgroup: Establecemos el encabezado de un grupo de columnas.

    colspan: Determina el número de columnas que ocupa la celda

    rowspan: Determina el número de filas que ocupa la celda.

    		 th
    td Permite definir una celda que contiene datos

    colspan: Determina el número de columnas que ocupa la celda

    rowspan: Determina el número de filas que ocupa la celda.
